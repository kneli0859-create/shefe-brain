# Conversion Copy — playbook

## Frameworks

- **AIDA** — Attention, Interest, Desire, Action
- **PAS** — Problem, Agitate, Solution
- **FAB** — Features → Advantages → Benefits

Pick one per section. Don't mix them in the same paragraph.

## Headline tests

Always write 3 variants and pick the one that:
1. Has a specific number (`30 Sekunden`, `12 Branchen`, `€197`)
2. Names the unwanted state (`statt 30 Minuten Excel`)
3. Reads aloud in < 5 seconds

## German specifics

- **Sie form** for B2B, always
- Substantivkomposita are fine if they're real words (`Reinigungskalkulator`)
- Avoid imported buzzwords (`game-changer`, `revolutionär` — overused)
- Use **konkrete Zahlen**: "60% schneller" not "viel schneller"

## CTA copy

- Action verb first: "Jetzt starten", "Demo ansehen"
- Pair with cost/risk reducer: "ab €197", "kostenlos · 2 Min"
- Avoid: "Klicken Sie hier", "Mehr erfahren"

## Email subject lines

- < 50 chars
- Personalize first 25 chars (highest signal in inbox preview)
- Curiosity > urgency for cold emails

## When to use

Any time you're writing customer-facing copy — landing, email, ads, PDF.
